דף נחיתה המסכם את הפיילוט, ובסופו שאלון משוב שהתשובות עליו נשלחות לגיליון Google Sheets.

שהדף ישמש בפועל.

## חיבור הטופס לגיליון אקסל (Google Sheets)

1. פתח/י גיליון Google Sheets חדש (או קיים) שאליו יגיעו התשובות.
2. בתפריט: **תוספים ← Apps Script** (Extensions ← Apps Script).
3. מחקו את הקוד שכבר קיים בקובץ, והדביקו במקומו את התוכן של הקובץ
   [`google-apps-script/Code.gs`](google-apps-script/Code.gs) מהתיקייה הזו.
4. שמרו את הפרויקט (סמל הדיסקט / Ctrl+S).
5. לחצו על **פריסה (Deploy) ← פריסה חדשה (New deployment)**.
   - סוג: **Web app**.
   - Execute as: **Me**.
   - Who has access: **Anyone**.
6. לחצו **Deploy**, אשרו הרשאות (זה הגיליון שלך, אז יש לאשר גישה).
7. העתיקו את ה-URL שמתקבל (מסתיים ב-`/exec`).
8. פתחו את הקובץ [`script.js`](script.js) בשורה הראשונה, ושימו את ה-URL במקום
   `PASTE_YOUR_APPS_SCRIPT_URL_HERE`.
9. שמרו, ו-commit + push לגיטהאב.

**חשוב:** בכל פעם שמעדכנים את הקוד ב-Apps Script יש ליצור **New deployment** מחדש (או
לעדכן deployment קיים דרך Manage deployments), אחרת השינויים לא ייכנסו לתוקף.

## הרצה מקומית

אפשר פשוט לפתוח את `index.html` בדפדפן.
